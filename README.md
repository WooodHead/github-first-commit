# github-first-commit


![screenshot](screenshots/buttons.png)

<p>
  Github的Commits页面只有Newer和Older两个按钮，但我经常需要查看最早的几个commit，所以我做了这个Chrome扩展，用于给Commits页面添加FirstPage和LastPage按钮。
</p>
<p>
  这是一个谷歌Chrome浏览器的扩展，请在Chrome的扩展程序管理页面启用“开发者模式”，然后“载入已解压的扩展”。
</p>
